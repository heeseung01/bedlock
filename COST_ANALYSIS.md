# 월간 예상 비용 분석 보고서

본 문서는 `bedlock` 애플리케이션을 AWS 환경에서 운영할 경우의 월간 예상 비용을 분석합니다.

## 분석 기준

- **리전**: 아시아 태평양(서울) `ap-northeast-2`
- **트래픽**: 개발 및 테스트 목적의 가벼운 트래픽
- **산정일**: 2026년 6월
